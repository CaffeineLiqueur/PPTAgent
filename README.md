# PPT Agent

一个智能 PPT 生成助手，通过自然语言交互自动创建精美的演示文稿。

## 项目简介

PPT Agent 是一个基于 AI 的演示文稿自动生成工具，能够理解用户的自然语言需求，自动生成结构化的 PPT 内容，并支持多种主题和样式定制。

## 功能特性

- 🤖 **智能内容生成**：基于 AI 理解用户需求，自动生成 PPT 内容大纲
- 📝 **自动排版**：智能布局和排版，确保视觉效果专业美观
- 🎨 **主题定制**：支持多种预设主题和自定义样式
- 📊 **图表支持**：自动生成数据图表和可视化内容
- 🔄 **实时预览**：支持实时预览和编辑
- 💾 **多格式导出**：支持导出为 PPTX、PDF 等格式
